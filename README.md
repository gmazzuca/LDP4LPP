# Discrete and Continuous Muttalib–Borodin Process: Large Deviations and Limit Shapes

This repository contains the numerical implementations and symbolic computation notebooks accompanying the paper **"Discrete and Continuous Muttalib–Borodin process: Large deviations and Riemann–Hilbert analysis"** by Jonathan Husson, Guido Mazzuca, and Alessandra Occelli.

The code evaluates the exact, closed-form analytical solutions derived via the Riemann-Hilbert Problem (RHP) analysis in the paper. It computes the equilibrium measures, plots the limit shapes of the associated plane partitions, and generates high-resolution heatmaps tracking the macroscopic phase transitions (subcritical to supercritical) and arctic curves.

## 📂 Repository Structure

* **`figures/`** Contains all the high-resolution PDF figures generated by the scripts and featured in the manuscript.
* **`PythonCode/`** Contains the three main, highly-optimized Python scripts used for the numerical evaluations and plotting:
  * `Simple_density.py`: Computes and plots the 1D equilibrium measure $\mu(x)$ across the different parameter regimes, seamlessly handling the exact analytical collapse in the supercritical (saturated) phase.
  * `heatmap_LDP4LPP.py`: Generates the 2D density heatmaps in the $(x, \xi)$ plane, overlaying the theoretical trajectories of the unconstrained edge ($x_b$) and the physical upper constraint ($x_{\max}$) to visualize the arctic curve.
  * `PlanePartition.py`: Maps the density back to the macroscopic spatial coordinate $\lambda$ (where $x = e^{-\beta\lambda}$) to plot the filling fraction and height profile of the plane partition limit shape.
* **`MathematicaCode/`** Contains the Mathematica notebooks used during the theoretical derivation to solve the algebraic systems of equations for the RHP moduli ($s_1, s_2, c_0, c_1$) and to generate early preliminary plots.
* **`Oldpythoncode/`** An archive of previous iterations of the numerical solvers (e.g., relying heavily on non-linear root finding before the exact analytical collapse was proved). Kept for reference and reproducibility.

## ⚙️ Prerequisites

To run the Python scripts, you will need **Python 3.8+** and the following standard scientific libraries:

```bash
pip install numpy scipy matplotlib
```
## 🚀 Usage
The Python scripts are self-contained and require no special command-line arguments. They are pre-configured with the specific parameter sets $(\gamma^2, \eta, \theta, \beta)$ used to generate the grids in the paper.
